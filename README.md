📚 [Документация Timeweb Cloud Apps](https://timeweb.cloud/docs/apps)

## <a name="dev"></a>Локальный запуск проекта
```bash
# установка зависимостей
cargo fetch

# запуск в дев режиме
cargo run

# сборка для продакшн
cargo run --release
```
